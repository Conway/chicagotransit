chicagotransit
==============

A collection of wrappers to communicate with various chicago transit APIs and infosources.

Installation
------------

To install, use ``pip install chicagotransit``

Currently Supported
-------------------

- CTA Bustracker (Bustime)

Future Expansion
----------------

- CTA Traintracker
- Divvy Bikes (WIP)
- Pace Busses*
- Metrarail*

* if some sort of public API can be found
